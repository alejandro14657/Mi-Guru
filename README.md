# Descripcion del proyecto
En el proyecto se trata sobre la implementacion del proyecto de capacitacion "Mi Guru", utlizando el lenguaje java, la IDE "Neatbeans" y utlizando la arquitectura de Micronúcleo (Microkernel) para el desarrollo. 

Este proyecto se trata sobre El software "Mi Gurú" el cual busca resolver el problema de la falta de capacitación efectiva y eficiente para los empleados de Tec Avanzada SAPI de CV. Porque actualmente, la empresa enfrenta altos costos debido a errores en la producción y un tiempo de capacitación prolongado, lo que afecta la productividad y la satisfacción laboral. Siendo este software una solucion posible para este problema.
